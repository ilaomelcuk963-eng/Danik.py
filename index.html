Задание 1: Начало работы с Python

В этом задании вы напишете свою первую программу на Python. Классическая программа "Hello, World!" является традиционным первым шагом в изучении любого языка программирования.

Вам нужно написать код, который выводит на экран фразу "Hello, World!".

Пример вывода:

Hello, World!

Для выполнения задания используйте функцию print(), которая выводит текст на экран.

Показать подсказку

Подсказка: В Python для вывода текста на экран используется функция print(). Текст, который нужно вывести, помещается в скобки и заключается в кавычки. Например:

print("Текст для вывода")

Попробуйте написать аналогичный код, но с текстом "Hello, World!".
